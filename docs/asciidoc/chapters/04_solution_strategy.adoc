== Solution Strategy

Die Lösungsstrategie basiert auf einer klaren Trennung zwischen der Steuerlogik (Python) und der visuellen bzw. physikalischen Darstellung (Unity).

---

=== Architekturelle Leitprinzipien

1. **Einheitliche Abstraktion**
   Der Benutzer soll nicht unterscheiden müssen, ob er die Simulation oder die reale Hardware steuert.
   Alle modusspezifischen Unterschiede (`virtual`, `real`, `both`) werden intern im System gekapselt.

2. **Entkopplung durch Netzwerkschnittstellen**
   Durch die Verwendung von UDP laufen Steuerlogik (Python) und Simulation (Unity) unabhängig voneinander.
   Dadurch wird verhindert, dass Performance-Schwankungen der 3D-Engine die Steuerlogik beeinflussen.

3. **Zustandsbasierte Kommunikation**
   Jedes UDP-Paket enthält den vollständigen Soll-Zustand (z. B. alle Gelenkwinkel).
   Dadurch kann ein verlorenes Paket durch das nächste automatisch korrigiert werden.

---

=== Entscheidungen zur Kommunikation

* **Warum UDP statt TCP?**
  In diesem System ist eine geringe Latenz wichtiger als eine garantierte Zustellung.
  Ein verlorenes Paket ist unkritisch, da kurze Zeit später ein neues Paket mit aktuellen Daten gesendet wird.
  TCP würde durch Wiederholungen (Retransmissions) zu Verzögerungen führen.

* **Warum JSON?**
  JSON ist einfach zu implementieren und gut lesbar, was das Debugging erleichtert.
  Da die Datenmenge pro Nachricht klein ist, überwiegen diese Vorteile gegenüber binären Formaten.

=== Strategie zur Welt-Manipulation

Die Verwaltung der Simulationswelt (z. B. Erzeugen oder Löschen von Objekten) wurde bewusst von der Robotersteuerung getrennt.

Während Roboterbewegungen als kontinuierlicher Datenstrom betrachtet werden, sind Änderungen an der Welt einzelne, diskrete Aktionen.

Aus diesem Grund wird für die Welt-Manipulation ein separater UDP-Port (5006) verwendet, während die Robotersteuerung über einen eigenen Kanal (5005) läuft.

Diese Trennung verhindert, dass grössere Nachrichten für Weltänderungen die kontinuierliche Übertragung von Bewegungsdaten beeinflussen und sorgt für eine klarere Struktur der Kommunikation.
